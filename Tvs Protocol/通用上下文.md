### 位置信息上下文
```json
{
	"header": {
		"namespace": "LocationInformation",
		"name": "LocationState"
	},
	"payload": {
		"latitude" : 30.0000,
		"longitude": 90.0000
	}
}
```

***Payload Paramters***

|	Parameter				|	Type		|	必选	|	描述				|
|	:-----------------------	|	:--------	|	:-----	|	:---------------	|
|	latitude					|	float		|	Yes	|	纬度				|
|	longitude				|	float		|	Yes	|	经度				|
